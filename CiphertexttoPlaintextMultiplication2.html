D-4.5 Ciphertext-to-Plaintext Multiplication

BGV’s ciphertext-to-plaintext multiplication scheme is exactly the same as BFV’s scheme (Summary D-2.6 in §D-2.6).

⟨Summary D-4.5⟩ BGV Ciphertext-to-Plaintext Multiplication

𝖱𝖫𝖶𝖤S,σ(M + ΔE) ⋅Λ

= (A, B) ⋅Λ

= (A ⋅Λ, B ⋅Λ)

= 𝖱𝖫𝖶𝖤S,σ((M ⋅Λ) + ΔE ⋅Λ)

Notice that BGV’s ciphertext-to-plaintext multiplication does not consume any multiplicative level.
