B-5.2 Decryption

We decrypt the first GLWE ciphertext (i = 1) using the secret S, with the scaling factor Δ1 = q β. This is because while the ciphertext contains l encryptions, the higher indices i > 1 have progressively smaller scaling factors Δi = q∕βi. If Δi becomes smaller than the noise threshold, those specific components cannot be decrypted correctly.
