A-8.1 Definitions

⟨Definition A-8.1⟩ Cyclotomic Polynomial

The 𝐧-th Cyclotomic Polynomial: is a polynomial whose roots are the primitive n-th roots of unity, that is:

Φn(x) = ∏ ζ∈P(n)(x−ζ) = ∏ 0≤k≤n−1, gcd(k,n)=1 (x−ωk)......, where ω = e2πi∕n

Remember the Euler’s formula: e2kπi∕n = cos⁡ (2kπ n ) + i ⋅sin⁡ (2kπ n )

A few pre-computed cyclotomic polynomials are as follows:

Φ1(x) = x −1
Φ2(x) = x + 1
Φ3(x) = x2 + x + 1
Φ4(x) = x2 + 1
Φ5(x) = x4 + x3 + x2 + x + 1
Φ6(x) = x2 −x + 1
Φ7(x) = x6 + x5 + x4 + x3 + x2 + x + 1
Φ8(x) = x4 + 1
Φ9(x) = x6 + x3 + 1
Φ10(x) = x4 −x3 + x2 −x + 1

As one example,

Φ4(x) = ∏ 0≤k≤3, gcd(k,4)=1 (x−ωk) = (x−ω1)(x−ω3) = (x−e2πi∕4)(x−e2⋅3πi∕4) = (x−eπi∕2)(x−e3πi∕2)

= (x − (cos⁡ (π 2) + i ⋅sin⁡ (π 2))) ⋅ (x − (cos⁡ (3π 2 ) + i ⋅sin⁡ (3π 2 )))

= (x −i)(x + i) = x2 + 1
