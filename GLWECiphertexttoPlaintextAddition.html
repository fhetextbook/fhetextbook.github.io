C-2 GLWE Ciphertext-to-Plaintext Addition

Suppose we have a GLWE ciphertext ct and a new plaintext polynomial Λ as follows:

𝖼𝗍 = 𝖦𝖫𝖶𝖤S,σ(ΔM + E) = (A0,A1,…,Ak−1,B) ∈R⟨n,q⟩k+1

Λ: a new plaintext polynomial

ΔΛ: a Δ-scaled new plaintext polynomial

Let’s define the following ciphertext-to-plaintext addition operation:

𝖼𝗍 + ΔΛ = (A0, A1,…,Ak−1, B + ΔΛ)

Then, the following is true:

⟨Summary C-2⟩ GLWE Homomorphic Addition with a Plaintext

𝖦𝖫𝖶𝖤S,σ(ΔM + E) + ΔΛ

= ({Ai}i=0k−1, B) + ΔΛ

= ({Ai}i=0k−1, B + ΔΛ)

= 𝖦𝖫𝖶𝖤S,σ(Δ(M + Λ) + E)

This means that adding a (Δ-scaled) plaintext polynomial Λ to a GLWE ciphertext that encrypts M and decrypting it yields M + Λ.

Proof.

1.
Since B = ∑ i=0k−1(Ai ⋅Si) + Δ ⋅M + E,

B + Δ ⋅Λ = ∑ i=0k−1(Ai ⋅Si) + Δ ⋅M + E + Δ ⋅Λ = ∑ i=0k−1(Ai ⋅Si) + Δ ⋅(M + Λ) + E

This means that (A0,A1,... Ak−1,B + ΔΛ) form the ciphertext 𝖦𝖫𝖶𝖤S,σ(Δ(Λ + M) + E)

2.
Thus,
𝖦𝖫𝖶𝖤S,σ(ΔM + E) + ΔΛ
= (A0, A1,... Ak−1, B + ΔΛ)
= ({Ai}i=0k−1, B + ΔΛ)
= 𝖦𝖫𝖶𝖤S,σ(Δ(M + Λ) + E)
□

Noise Growth: Note that after decryption, the original ciphertext 𝖼𝗍 + ΔΛ’s noise E stays the same as before. This means that ciphertext-to-plaintext addition does not increase the noise level.
