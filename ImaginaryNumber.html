A-11.2 Imaginary Number

⟨Definition A-11.2⟩ Imaginary Number

  • i is defined to be an imaginary number that has the property: i2 = −1
  • Complex Number is a number that involves imaginary number(s) (e.g., a = 5.6 + i ⋅ 4.3)
  • Real Number is a number that does not involve any imaginary number (e.g., a = 13.4)

  • a¯ is a Conjugate of a if a and a¯ have the same real number part and an opposite-signed imaginary number part

    (e.g., a = 3 + i ⋅ 3.4, a¯ = 3 − i ⋅ 3.4)

  • Hermitian Vector is a vector where the 1st half of its elements is a conjugate of the 2nd half, as the n-dimensional vector illustrated below:

    v→ = (v1,v2,v3,⋯,vn 2 −1,vn 2 ,v¯n 2 ,v¯n 2 −1,⋯,v¯3,v¯2,v¯1)
