A-1.3 Inverse

⟨Definition A-1.3⟩ Inverse in Modulo Arithmetic

In modulo q (i.e., in the world of remainders where all numbers have been divided by q), for each a ∈{0,1,2,⋯,q −1}:

  • Additive Inverse of a is denoted as a+−1 which satisfies a + a+−1 ≡0 mod q. For example, in modulo 11, 3+−1 = 8, because 3 + 8 ≡0 mod 11.
  • Multiplicative Inverse of a is denoted as a∗−1 which satisfies a ⋅a∗−1 ≡1 mod q. Such an inverse exists if and only if gcd ⁡ (a,q) = 1. For example, modulo 11, 3∗−1 = 4, because 3 ⋅4 ≡1 mod 11.
