A-4.2 Theorems

⟨Theorem A-4.2.1⟩ Order Property (I)

For a ∈ 𝔽×, and n ≥1, an = 1 if and only if ord𝔽(a) ∣ n

(i.e., 𝗈𝗋𝖽𝔽(a) divides n).

Proof.

1.
Forward Proof: If 𝗈𝗋𝖽𝔽(a) | n, then for 𝗈𝗋𝖽𝔽(a) = k where k is a’s order, and n = 𝑙𝑘 for some integer l.

Then, an = a𝑙𝑘 = (ak)l = 1l = 1.

2.
Backward Proof: If an = 1 and 𝗈𝗋𝖽𝔽(a) = k, write n = 𝑞𝑘 + r with 0 ≤r < k. Then 1 = an = a𝑞𝑘+r = (ak)qar = ar. By minimality of k, we must have r = 0, hence k∣n.
□

⟨Theorem A-4.2.2⟩ Order Property (II)

If 𝗈𝗋𝖽𝔽(a) = k, then for any n ≥1, 𝗈𝗋𝖽𝔽(an) = k gcd ⁡ (k,n).

Proof.

1.
ak,a2k,a3k,… = 1.
2.
Given 𝗈𝗋𝖽𝔽(an) = m, (an)m,(an)2m,(an)3m,… = 1
3.
Note that by definition of order, x = k is the smallest value that satisfies ax = 1. Thus, given 𝗈𝗋𝖽𝔽(an) = m, then m is the smallest integer that makes (an)m = 1. Note that (an)m should be a multiple of ak, which means 𝑚𝑛 should be a multiple of k. The smallest possible integer m that makes 𝑚𝑛 a multiple of k is m = k gcd ⁡ (k,n).
□

⟨Theorem A-4.2.3⟩ Order Property (III)

𝗈𝗋𝖽𝔽(a) = 𝑘𝑛 if and only if 𝗈𝗋𝖽𝔽(ak) = n.

Proof.

1.
Forward Proof: Given 𝗈𝗋𝖽𝔽(a) = 𝑘𝑛, and given Theorem A-4.2.2, 𝗈𝗋𝖽𝔽(ak) = 𝑛𝑘 gcd ⁡ (k,𝑛𝑘) = 𝑛𝑘 k = n.
2.
Backward Proof: Given 𝗈𝗋𝖽𝔽(ak) = n and letting 𝗈𝗋𝖽𝔽(a) = m, Theorem A-4.2.2 gives m∕gcd ⁡ (m,k) = n, so m = ngcd ⁡ (m,k). In particular k∣m, hence m = 𝑛𝑘.
□

⟨Theorem A-4.2.4⟩ Fermat’s Little Theorem

Given |𝔽|= p (a prime) and a ∈ 𝔽, ap = a.

Proof.

1.
If a = 0, then ap = a = 0.
2.
If a≠0, then a ∈ 𝔽×, the multiplicative group of the field, which has size |𝔽×|= p −1. By Lagrange’s theorem (in a finite group G, the order of any element divides |G|), the order of a divides p −1, hence ap−1 = 1. Therefore ap = a.
□
