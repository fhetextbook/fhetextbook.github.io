A-10.1 Vector Arithmetic

This section explains the basic arithmetic of vector and matrix computations, as well as advanced concepts such as vector/plane projection and the basis of planes (or spaces).

⟨Definition A-10.1⟩ Vector Arithmetic

  • Addition: Given two n ×1 vectors (i.e., n-dimensional vectors) composed of n numbers each:

    a→ = (a0,a1,⋯,an−1), b→ = (b0,b1,⋯,bn−1)

    Vector addition is defined as:

    a→ + b→ = (a0 + b0, a1 + b1, ⋯, an−1 + bn−1)

  • Dot Product: Given two n-dimensional vectors:

    a→ = (a0,a1,…,an−1), b→ = (b0,b1,…,bn−1)

    Dot product is defined as:

    ⟨a→,b→⟩= a→ ⋅b→ = a0b0 + a1b1 + ⋯ + an−1bn−1.

    a→ ⋅b→ can also be expressed as ∥a→∥∥b→∥cos⁡𝜃, where 𝜃 is the angle between a→ and b→. If a→ and b→ point in the same direction, a→ ⋅b→ attains its maximum value (∥a→∥∥b→∥). If a→ and b→ are orthogonal, then a→ ⋅b→ = 0.

  • Hadamard Product: Given two n-dimensional vectors:

    a→ = (a0,a1,…,an−1), b→ = (b0,b1,…,bn−1)

    Hadamard product is defined as:

    a→ ⊙b→ = (a0b0, a1b1, ⋯, an−1bn−1)

  • Hermitian Product: Given two n-dimensional complex vectors:

    a→ = (a0 + i ⋅a0′, a1 + i ⋅a1′, …, an−1 + i ⋅an−1′)

    b→ = (b0 + i ⋅b0′, b1 + i ⋅b1′, …, bn−1 + i ⋅bn−1′)

    Hermitian product is a dot product with the 2nd operand as a conjugate (§A-11.2):

    ⟨⟨a→,b→⟩⟩= a→ ⋅b→¯

    = (a0 + i ⋅a0′, a1 + i ⋅a1′, ⋯, an−1 + i ⋅an−1′) ⋅(b0 −i ⋅b0′, b1 −i ⋅b1′, ⋯, bn−1 −i ⋅bn−1′)
